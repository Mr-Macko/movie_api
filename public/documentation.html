Movie API

This API will provide a database of movies. It will support searches by director and genre.

Business Logic URL HTTP Method Request body data format Response body data format
Return a list of ALL movies to the user /movies GET None A JSON object holding data about all movies
Return data about a single movie by title to the user /movies/:title GET None A JSON object holding data about a single movie, containing title, genre, actor and director.
Example:
{
"Genre": {
"Name": "Action",
"Description": "Action is a film genre of entertainment cinema in which the progress of the external plot is driven forward and illustrated by mostly spectacularly staged scenes of combat and violence."
},
"Director": {
"Name": "Christopher Nolan",
"Bio": "Christopher Edward Nolan is a British-American film director, producer, and screenwriter.",
"Birth": "1970-07-30T00:00:00.000Z"
},
"_id": "61b27223d8b1a67422eb3e7d",
"MovieID": "0004",
"Title": "Inception",
"Description": "A thief steals corporate secrets using shared dreaming technology. Then he gets an order to fix an idea in the head of a CEO.",
"ImagePath": "inception.png",
"Featured": false
}
Return data about a genre by name /movies/genre/:name GET None A JSON holding data about a single genre.
{
"name": "Genre Name",
"description": "Genre description",
},
Return data about a director by name /directors/:name GET None A JSON holding data about a single director.
Example
{
"Name": "Christopher Nolan",
"Bio": "Christopher Edward Nolan is a British-American film director, producer, and screenwriter.",
"Birth": "1970-07-30T00:00:00.000Z"
}
Allow new users to register /users POST A JSON holding data about a new user.
Example
{
"Username": "Test1",
"Password": "TestPassword",
"Email": "testmail@gmail.com"
} 		A JSON holding data about a new created user with ID.

"Password": "TestPassword",
"Email": "testmail@gmail.com",
"FavortieMovies": [],
"_id": "61b7d0cd620859ec2da02a4c",

Allow users to update their user info /users/:username PUT A JSON holding updated data of a user
Example:
"Password": "asdf12gfdgdfgdfgdfg34", 		A JSON holding updated data of the user.

Allow users to add a movie to their list of favorites /users/:username/movies/:movieid PUT None A text message indicating whether a movie was successfully added to favorites list.
Allow users to remove a movie from their list of favorites /users/:username/movies/:movieid DELETE None A text message indicating whether a movie was successfully deleted to favorites list.
Allow existing users to deregister /users/:username DELETE None A text message indicating whether a user was successfully deleted tfrom the database.
