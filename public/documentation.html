Movie API

This API will provide a database of movies. It will support searches by actor and genre.

Business Logic URL HTTP Method Request body data format Response body data format
Return a list of ALL movies to the user /movies GET None A JSON object holding data about all movies
Return data about a single movie by title to the user /movies/:title GET None A JSON object holding data about a single movie, containing title, genre, actor and director.
Documentation {
"title": "Movie Title",
"genre": "Genre Name",
"actor": "Actor Name"
"director": "Director Name"
},
Return data about a genre by name /movies/genre/:name GET None A JSON holding data about a single genre.
{
"name": "Genre Name",
"description": "Genre description",
},
Return data about a director by name /directors/:name GET None A JSON holding data about a single director.
{
"name": "Director Name",
"bio": "Director Bio",
"birth-year": "Birth Year",
"death-year": "Death Year",
},

Allow users to add a movie to their list of favorites /user/:username/movies/:favorite-movies PUT None A text message indicating whether a movie was successfully added to favorites list.
Allow users to remove a movie from their list of favorites /user/:username/movies/:favorite-movies DELETE None A text message indicating whether a movie was successfully deleted to favorites list.
Allow existing users to deregister /users/:userID DELETE None A text message indicating whether a user was successfully deleted tfrom the database.
